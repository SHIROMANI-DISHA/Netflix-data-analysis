# Netflix-data-analysis

## About the Project
This project is a basic data analysis of Netflix’s movies and TV shows dataset. The goal was to explore the data, understand patterns, and see how insights can be used to support decisions.

I worked on cleaning the dataset, handling missing values, and then analyzing trends using visualizations.

---

## What I Did

- Cleaned the dataset by handling missing values in important columns like country, rating, and date_added
- Explored the structure of the data to understand its limitations
- Created visualizations to study:
  - Distribution of Movies vs TV Shows
  - Top content producing countries
  - Growth of content over the years
  - Ratings distribution
  - Popular genres
- Built a simple machine learning model to understand relationships between content type and rating

---

## Key Observations

- Movies are more common than TV shows on the platform
- Most content comes from the USA, followed by India and a few other countries
- There is a noticeable increase in content after 2015
- A large portion of content is targeted at mature audiences
- Certain genres like Drama and International content appear more frequently

---

## Limitations

- Some columns like 'director' and 'cast' had many missing values, so they were not used in detailed analysis
- Country-based analysis only considers available data (rows with missing country were removed)
